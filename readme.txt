BombJack PSP Remake is here! Full game with 14 levels. You can edit and modify
with Windows Level Editor. New features:

- Improvements in game engine
- Walls add
- Extra life add


Controls
------------------------------------------------------------------------------

In main menu:

Circle: Show top scores
Triangle: Show Credits
Cross: Return to menu 

In game

Circle: Jump
Start: Pause
Left & Right to move Jack


Installation
------------------------------------------------------------------------------

Unpack, copy folder to PSP and overwrite all
 
 
Levels
------------------------------------------------------------------------------

14 full levels
If you want to modify levels or create new ones, use only Windows Level editor 
1.0, available shortly. You can change all elements, included backgrounds!
Backgrounds must be PNG files with same size of originals.

Levels by Peligros


Firmware Compatibility
